# Xiao Ouyang - Academic Homepage

Personal academic homepage for Xiao Ouyang (欧阳霄), 3rd year CS undergraduate at Tsinghua University.

## Setup

1. Replace `assets/profile.jpg` with your headshot photo
2. Replace `assets/rdt2-thumb.jpg` with a figure from the RDT-2 paper
3. Add your CV as `assets/Xiao_Ouyang_CV.pdf`
4. Update the email and Google Scholar link in `index.html`
5. Push to GitHub and enable GitHub Pages

## Deployment

This site is designed to work with GitHub Pages. Create a repository named `ouyangxiao23.github.io` and push these files to the `main` branch.

Site URL: https://ouyangxiao23.github.io/
